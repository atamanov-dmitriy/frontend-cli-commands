### Справочная информация

|                                                     |                     |
| --------------------------------------------------- | ------------------- |
| Проверка установки git                              | `git -v`            |
| Посмотреть ссылки на удаленный репозиторий (origin) | `git remote -v`     |
| Показывает изменения и их статус                    | `git status`        |
| Показывает коммиты (Аналог сейвов из игр)           | `git log --oneline` |
| Проверка установки Node package manager             | `npm -v`            |
| Проверка установки Node                             | `node -v`           |

### Подготовить сохранение

|                                  |                                                                      |
| -------------------------------- | -------------------------------------------------------------------- |
| Подготовить к сохранению         | `git add .`                                                          |
| Сохранить<br>Пример команды выше | `git commit -m "<Имя коммита>"`<br>`git commit -m "my-first-commit"` |
|                                  |                                                                      |

### Работа с удаленными репозиториями

|                                                                          |                                                                                                                                                             |
| ------------------------------------------------------------------------ | ----------------------------------------------------------------------------------------------------------------------------------------------------------- |
| Отрпавить коммиты (сейвы) на удаленны репозиторий<br>Пример команды выше | `git push -u <имя-удаленного-репозитория> <название-ветки-удаленного-репозитория>`<br>`git push -u origin main`                                             |
| Изменить ссылку на удаленный репозиторий<br>Пример команды выше          | `git remote set-url <имя-репозитория> <url-ссылка на репозиторий>`<br>`git remote set-url origin https://github.com/atamanov-dmitriy/frontend-cli-commands` |
| Получить коммиты (сейвы) из удаленного репозитория                       | `git pull`                                                                                                                                                  |
| Склонировать удаленный репозиторий на локальный компьютер                | `git clone <url-ссылка на репозиторий>`                                                                                                                     |
| Пример команды выше                                                      | `git clone https://github.com/atamanov-dmitriy/react-project-1.git`                                                                                         |

### Работа с приложением

|                                                                               |                                                                        |
| ----------------------------------------------------------------------------- | ---------------------------------------------------------------------- |
| Открыть приложение в VS Code                                                  | `code .`                                                               |
| Показывает доступные команды                                                  | `npm run`                                                              |
| Устанавливает необходимые пакеты                                              | `npm ci` (или `npm i`)                                                 |
| Создать проект на React<br>Пример команды выше                                | `npm init react-app <название-проекта>`<br>`npm init react-app my-app` |
| Запустить проект<br>Запущенный проект будет доступен по адресу localhost:3000 | `npm start` (или `npm run start`)                                      |

### PowerShell и Bash

|                                                                       |                                                               |
| --------------------------------------------------------------------- | ------------------------------------------------------------- |
| Если возникает ошибка "Выполнение сценариев отключено в этой системе" | `Set-ExecutionPolicy RemoteSigned -Scope LocalMachine -Force` |
| Сменить директорию (от Change directory)                              | `cd`                                                          |
| показать доступные для перехода каталоги                              | `cd` + нажатие _Tab_                                          |
| Показать каталоги                                                     | `ls`                                                          |
| Отменить работающую команду                                           | _Ctrl_ + _C_                                                  |
